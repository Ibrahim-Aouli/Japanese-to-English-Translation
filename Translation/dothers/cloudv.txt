To use cloud save
--------------------------------------------------End--

Make sure PlayStation{Registered}3 and PlayStation{Registered}Vita are signed into the same PSN account.
Then you can store,share and synchronize saves over PlayStation{Registered}Net work between 
them by linking saves to cloud server.
--------------------------------------------------End--

To synchronize trophies
--------------------------------------------------End--

Make sure PlayStation{Registered}Network account has been signed in,then system will share and 
synchronize trophies when using data transfer.
